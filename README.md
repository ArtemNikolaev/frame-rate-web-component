# FrameRate WC
> WebComponent which shows the current page frame rate.
> Can be easy added on any html-page.

## Installation
####Prerequarement
  * [NodeJS](https://nodejs.org/en/)

#### Prerequisite
```
    npm install
```    

#### Run
```angular2html
    // dev
    npm start

    // prod
    npm build
```
## Documentation

#### On / Off functionality
Use attribute `off` to disable component.
```angular2html
    <frame-rate off></frame-rate>
```
